# Magic Fox Game

## About the Game

<p>This game is based on a tutorial from the YouTube channel <a
                                href="https://www.youtube.com/c/Frankslaboratory">Franks laboratory</a>. I drew the
                            spritesheets myself, but most oft them are based on the spritesheets provided in the
                            tutorial, which are created by the awesome artist <a href="https://bevouliin.com/">
                                bevouliin</a>.</p>
